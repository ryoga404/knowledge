# ナレッジデータベースアプリ BETA

## 開発目的
今までのエラー対処やパッケージインストールの方法などを検索できるようしたい＋どのデバイスからでもアクセス、共有できるようにするため作成（現在、Linuxのみ）

## 開発環境
Flutter SDK
Android Studio
Dart
SQLite
Claude Pro（一部バイブコーディングで作成）

## 機能

１、フォルダ分け機能
任意の項目ごとにデータを分けることができる。
